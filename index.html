How to Use

Initialize State

  • Complex Vector: Enter 4 amplitudes for |00⟩, |01⟩, |10⟩, |11⟩ (e.g., "1", "0", "0", "0" for |00⟩)
  • Full 2-Qubit: Enter α, β for Qubit 1 and γ, δ for Qubit 0

Apply Gates

  • Select Quantum Gates tab
  • Click predefined gate (CNOT, CZ, iSWAP, H, X, Y, Z) or enter custom 4×4 unitary matrix
  • Click Decompose to see physical implementation

View Results

  • Bloch Spheres: 3D visualization of each qubit state (drag to rotate)
  • Probabilities: Measurement probabilities |c₁|², |c₂|², |c₃|², |c₄|²
  • Operations: Physical gate decomposition with drive parameters
