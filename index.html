How to Use

Initialize State

  • Complex Vector: Enter 4 amplitudes for |00⟩, |01⟩, |10⟩, |11⟩
  • Full 2-Qubit: Enter α, β for Qubit 1 and γ, δ for Qubit 0

Apply Gates

  • Select the Quantum Gates tab
  • Click a predefined gate or enter a custom 4×4 unitary matrix
  • Click Decompose to see the physical implementation

View Results

  • Bloch Spheres: 3D visualization of each qubit state
  • Probabilities: Measurement probabilities for each basis state
  • Operations: Physical gate decomposition with drive parameters
